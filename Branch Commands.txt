1.Display a list of all branches on the current remote {
	git branch
}

2. Display a list of all branches from all remotes {
	git branch -r
}

3. Switch to a branch {
	git checkout <branchname>
}

4. Make a new branch and switch to it {
	git checkout -b <branchname>
}

5. Push the branch {
	git push -u origin <branchname>
}

6. Delete a branch {
	git branch -d <branchname>
}
